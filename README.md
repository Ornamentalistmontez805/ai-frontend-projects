## 🚀 Quick Start

**Option 1 — Clone and open in browser:**
```bash
git clone https://github.com/SNTL84/ai-frontend-projects.git
cd ai-frontend-projects
open masteraifrontendroadmap.html   # Interactive roadmap
open api-projects/01-chat-interface/index.html
```

**Option 2 — Pick a project, add your API key:**

Each project uses a single `index.html`. Open it, find:
```javascript
const API_KEY = 'YOUR_API_KEY_HERE';
```
Replace with your key, save, open in browser. That's it.
