## ⚙️ Setting up your environment

You need a runtime environment to execute the project code. Follow these steps to prepare your computer.

1. Download the latest version of Node.js for Windows from the official website.
2. Run the installer and keep all default settings.
3. Restart your computer after the installation finishes.
4. Open the Windows "Command Prompt".
5. Type `node -v` and press Enter. If you see a version number, your setup works.

## 🚀 Running your first project

Each project contains a "README.txt" file inside its folder. You must follow these steps to see the project in your browser:

1. Open the Command Prompt.
2. Type `cd` followed by a space, then drag the folder for your chosen project into the window. Press Enter.
3. Type `npm install` and press Enter. Wait for the green progress bars to finish.
4. Type `npm run dev` and press Enter.
5. Wait for the terminal to show a web address, usually "http://localhost:5173".
6. Copy that address and paste it into your web browser.

## 🔑 Using your own API keys

These projects connect to external services. You need an API key to make them talk to the AI models.

1. Sign up for an account at the provider website, such as OpenAI or Claude.
2. Generate an API Key in your account settings.
3. Look for a file named ".env.example" in your project folder.
4. Copy this file and rename the new copy to ".env".
5. Open the ".env" file in Notepad.
6. Replace the placeholder text with your actual API key.
7. Save the file and restart your project in the terminal.

## ❓ Troubleshooting common issues

If you encounter errors, check these common points of failure:

*   Missing keys: Ensure your ".env" file contains valid keys without extra spaces.
*   Dependency errors: If `npm install` fails, delete the "node_modules" folder and run the command again.
*   Port conflicts: If the project won't start, try closing other web development windows or restart your computer.
*   Internet access: Ensure your firewall allows the Node.js application to request data from external AI servers.

## 📂 Understanding the project structure

Every project follows a consistent layout to help you navigate:

*   `/src`: Contains the primary code that controls the logic and look of the site.
*   `/public`: Holds images, fonts, and assets.
*   `/node_modules`: Keeps the supporting libraries needed for the build.
*   `package.json`: Lists the specific tools required for that project.
*   `.env`: Stores your private keys securely.
